## Projects Showcase

Below is a summary of the projects contained in this portfolio. **Click on any project title to navigate directly to that project's folder**, where you will find the final Excel workbook, a detailed `README.md`, and relevant screenshots.

### ➡️ [Capstone Project: End-to-End Excel Analysis](./Capstone_Project_Excel_Analysis/)
*   **Description:** A comprehensive, end-to-end analysis of a raw sales dataset. This project covers the full data analysis workflow, from data cleaning and preparation to building an interactive dashboard, performing strategic modeling with What-If tools, and conducting advanced statistical analysis with the Analysis ToolPak.
*   **Skills:** Data Cleaning, Excel Tables, PivotTables, Charting, Interactive Dashboards, Goal Seek, Solver, Regression Analysis.

### ➡️ [09. Mastering Lookup Functions](./09_Lookup_Mastery_Project/)
*   **Description:** A focused project demonstrating the essential skill of merging data from multiple sources. It showcases proficiency in modern and legacy lookup functions to enrich a dataset for analysis.
*   **Skills:** `XLOOKUP`, `VLOOKUP`, `INDEX/MATCH`.

### ➡️ [08. Statistical Analysis with the Analysis ToolPak](./08_Statistical_Analysis_ToolPak/)
*   **Description:** A deep dive into Excel's statistical capabilities. This project uses the Analysis ToolPak to perform descriptive statistics, hypothesis testing (ANOVA, t-Test), forecasting, and builds a predictive linear regression model.
*   **Skills:** Descriptive Statistics, ANOVA, t-Test, Correlation, Linear Regression, Moving Average.

### ➡️ [07. Optimization with Solver](./07_Solver_Mastery/)
*   **Description:** A portfolio of classic "Operations Research" problems solved using the Excel Solver Add-in. This project demonstrates how to find the optimal solution for complex business problems involving cost minimization, efficiency maximization, and budget allocation.
*   **Skills:** Linear Programming, Optimization, Constraint Modeling, Sensitivity Analysis.

### ➡️ [06. What-If Analysis for Strategic Modeling](./06_What-If_Analysis/)
*   **Description:** This project focuses on predictive and strategic modeling using Excel's What-If Analysis tools. It includes determining a product's break-even point with Goal Seek and comparing multiple business strategies using the Scenario Manager.
*   **Skills:** Goal Seek, Scenario Manager, Financial Modeling.

### ➡️ [05. Advanced Excel Tables Mastery](./05_Excel_Tables_Mastery/)
*   **Description:** A project dedicated to Excel Tables, covering structured references, dynamic calculations, the total row, and using tables as a robust source for PivotTables.
*   **Skills:** Excel Tables, Structured Referencing, Dynamic Ranges.

### ➡️ [04. Advanced PivotTable Mastery](./04_PivotTable_Mastery/)
*   **Description:** A comprehensive exploration of PivotTables, covering multi-level analysis, date grouping, calculated fields, and the use of `GETPIVOTDATA` to create dynamic summary reports.
*   **Skills:** PivotTables, PivotCharts, Slicers, Calculated Fields, `GETPIVOTDATA`.

### ➡️ [03. Interactive Dashboard Project](./03_Dashboard_Project/)
*   **Description:** This project focuses on data visualization and creating a user-friendly, interactive report. A complete dashboard was built from scratch, powered by a backend of PivotTables and controlled by Slicers.
*   **Skills:** Dashboard Design, Charting, PivotTables, Slicers, Report Connections.

### ➡️ [02. Filtering & Conditional Formatting](./02_Filtering_And_CF_Project/)
*   **Description:** A project demonstrating how to use Excel's filtering tools and conditional formatting rules to isolate, explore, and highlight key information within a dataset.
*   **Skills:** AutoFilter, Advanced Filter, Conditional Formatting (Formulas, Color Scales, Data Bars).

### ➡️ [01. Foundational Sorting Techniques](./01_Sorting_Project/)
*   **Description:** A project focused on the fundamentals of data organization in Excel, covering single-column, multi-level, and custom list sorting.
*   **Skills:** Sorting, Multi-Level Sorting, Custom Lists.
